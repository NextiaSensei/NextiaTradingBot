## ✨ Features

### 🤖 Core Capabilities
- **Multi-Strategy Execution** - Run multiple strategies simultaneously
- **Real-Time Market Analysis** - Live price feeds and technical indicators
- **Risk Management** - Automated position sizing and stop-loss management
- **Performance Analytics** - Real-time trade tracking and metrics

### 🔧 Technical Stack
- **Python 3.8+** - Core programming language
- **MetaTrader5 API** - Direct broker integration
- **pandas & numpy** - Data analysis and manipulation
- **python-dotenv** - Secure configuration management

## 🛠 Installation

### Prerequisites
- Python 3.8 or higher
- MetaTrader 5 installed
- Active MT5 demo/live account

### Quick Start
```bash
# Clone repository
git clone https://github.com/NextiaSensei/NextiaTradingBot.git
cd NextiaTradingBot

# Install dependencies
pip install -r requirements.txt

# Configure environment
cp .env.example .env
# Edit .env with your MT5 credentials

# Launch bot
python main.py
